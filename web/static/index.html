Dispatch

          

            

              
              
              
Fallback model when a request does not specify one.

            

            

              
              
              
How to pick a provider when more than one matches.

            

          

          

            
            
          

          

            

              
              
            

            

              

                
                
              

              

                
                
              

            

            
If a prompt matches any pattern, it's only sent to providers carrying this tag. If none is available the request is refused (never sent to cloud).

          

          

            

              
              
            

            

              

                
                
              

              

                
                
              

              

                
                
              

              

                
                
                
              

            

            
Heuristic classifier filters candidates by their tier:cheap / tier:standard / tier:strong tag. Tag your providers (Providers tab → Data → tags) to opt them into each tier. Empty result falls back to all candidates.

          

        


        

          
Budget

          

            
            
          

          

            

              
              
            

            

              
              
            

            

              
              
            

          

          
When enforced, /v1 returns 429 once a cap is reached (0 = unlimited). Warn just logs a server warning. Caps cover all traffic; per-key caps are set on each API key.

🕵️ MITM Proxy

      

        Local HTTPS interceptor for AI-coding IDEs (Antigravity, Copilot, Cursor, Kiro). Generates a per-machine root CA, auto-signs leaf certs per SNI, and rewrites IDE traffic through this router's dispatcher.

Root Certificate Authority

        

          The root CA must be installed in your OS trust store so IDE clients see valid certs for intercepted hosts. Install once per machine.
        

        

          ⬇ Download rootCA.pem
          
          
        

        

      


      

        
DNS hijack (hosts file)

        

          Append 127.0.0.1 entries for intercepted hosts so the OS routes them to this MITM listener. Idempotent — re-applying does nothing.
