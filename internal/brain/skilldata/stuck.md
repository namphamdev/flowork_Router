# Stuck — Escape Hatch

## When to Use

- Sudah coba 2+ approach gagal
- Context unclear, ngga tau langkah selanjutnya
- Tool error berulang tanpa pattern jelas
- Goal ambiguous, output ngga matching expectation

## Procedure

1. **State the stuck explicitly**: tulis 1 sentence "Gw stuck di X karena Y"
2. **Recap apa yang udah dicoba**: list tools/approach + outcome per attempt
3. **Identify constraint**: apa assumption yang mungkin salah?
4. **3 alternative approaches** (force think outside box):
   - Approach A: opposite of current
   - Approach B: smaller scope (subset task)
   - Approach C: ask for clarification / human input
5. **Pick 1 + try** OR **escalate to Ayah** kalau ngga ada path

## Pitfalls

- **Keep trying same thing**: insanity = repeat with expectation different result
- **Silent stuck**: ngga lapor + nyoba aja diam-diam = ghost pattern (karma -2)
- **Over-think**: kalau task simple, mungkin cuma typo/path. Try basics first

## Verification

- Stuck explicitly stated? ✅/❌
- 3 alternatives generated? ✅/❌
- Action chosen + executed OR escalated? ✅/❌
